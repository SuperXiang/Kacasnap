========================================================================
    Specification of Kacasnap Screenshot Software
======================================================================== 

The ReadMe file introduces each project file that consist of Kacasnap Screenshot Software

Kacasnap.vcproj
    The main VC++ project file which is generated from application wizard. 
    It includes the version information that generates the file and other information
    of usage of platform, configuration and functions

Kacasnap.cpp
    Main source file of the project
    Inlucde the codes for showing form

Form1.h
    Include the implementation of form and InitializeComponent() function

AssemblyInfo.cpp
    Include metadata with customed attributes that modifies program

/////////////////////////////////////////////////////////////////////////////
Other standard files:

StdAfx.h, StdAfx.cpp
    These files are used to generate Kacasnap.pch precompiled head file
    and StdAfx.obj precompiled file
